Server Analytics

Simple Commands

Everything is accessible through intuitive Discord slash commands.

/analytics stats View server statistics and activity overview
/analytics reports generate Generate a beautiful PDF analytics report
/analytics reports schedule Schedule automated daily, weekly, or monthly reports
/analytics heatmap View activity heatmap for your server
/analytics alerts Configure smart alerts for any metric
/analytics keywords Manage tracked keywords
/analytics streaks View user activity streaks
/analytics help View all available commands
