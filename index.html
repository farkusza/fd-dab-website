Commands

23+ Slash Commands

Everything is accessible through intuitive Discord slash commands. No config files, no YAML. Just type and go.

/dab stats Server statistics — messages, joins, leaves, peak hours, top contributors, retention
/dab insight AI-powered health insights with actionable recommendations for your server
/dab compare Compare server metrics: current 7 days vs previous 7 days
/dab leaderboard Top contributors ranked by engagement score (messages, reactions, voice, streaks)
/dab trends Growth trends with sparkline chart and week-over-week comparison
/dab heatmap Activity heatmap by day of week and hour
/dab pdf generate Generate a 7-day, 30-day, or custom PDF analytics report
/dab schedule Configure automated scheduled reports (daily/weekly/monthly)
/dab export csv Export analytics data as CSV for external analysis
/dab voice Voice channel usage stats and session data
/dab reactions Most-used emojis and reaction statistics
/dab roles Role activity rankings with active member counts
/dab streak Check your own activity streak
/dab ghostpings Recent ghost pings (deleted @mentions)
/dab alerts Set custom alert thresholds for any metric
/dab branding Customize PDF report branding (logo, colors, title)
/dab webhooks Configure webhook integrations for external services
/dab help View all available commands with descriptions
