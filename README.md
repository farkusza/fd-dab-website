## 🤖 Bot

FD-DAB gives your Discord server powerful analytics, beautiful PDF reports, and actionable insights.

### Features
- 📊 Server Analytics — Track messages, joins, leaves, reactions, voice activity
- 📈 PDF Reports — Beautiful branded analytics reports
- 🔥 Activity Heatmaps — Visualize when your community is most active
- 👥 User Streaks — Track consecutive daily activity
- 🔔 Smart Alerts — Custom thresholds for any metric
- 👻 Ghost Ping Detection — Never miss a mention
- 💬 Support Inbox — Receive feedback via email
